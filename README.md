## 📂 Folder Structure

*   **`Revision/`** – The core learning workspace containing daily practice files, basic syntax modules, and coding problem sets.
    *   `Practice-Set/` – Structured programming logic challenges and algorithmic practice scripts.
*   **`myenv/`** *(Hidden via .gitignore)* – A centralized, shared virtual environment used to optimize disk space while managing modern dependencies.

---

## 🛠️ Environmental Architecture & Setup

This repository is built using an optimized development workflow to ensure clean dependency tracking and robust local process management:

1.  **Isolated Sandboxing:** Utilizes a shared virtual environment (`venv`) to eliminate global package clutter and prevent version conflicts.
2.  **Visual Studio Code Tooling:** Customized to use native integrated terminal execution contexts instead of third-party wrappers, enabling full user interactivity (`input()` functions).
3.  **Strict Source Control:** Configured with specific `.gitignore` global wildcards to keep compiled caches (`__pycache__/`) and massive media files (such as `.mp3` and `.wav` tracks) isolated to the host machine.
